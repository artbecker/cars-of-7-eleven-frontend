# Cars of 7-Eleven - Frontend Project

A car photography sharing platform built with React, featuring Tumblr API integration, photo uploads, and social interactions.

## 🌟 Features

- **Photo Fetching**: Fetch tagged photos via API from Tumblr
- **Social Interactions**: Like photos (only visual, not updating the backend)
- **Responsive Design**: Optimized for desktop and mobile devices
- **User Info**: It is only mocked, for future updates with auth features

## 🛠️ Technologies Used

- **Frontend**: React.js, HTML5, CSS3
- **Routing**: React Router
- **API Integration**: RESTful API consumption
- **Styling**: CSS Modules/BEM methodology
